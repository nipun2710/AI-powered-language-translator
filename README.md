AI Multi-Language Translator 🌍🎙️

An AI-powered multilingual translation system built using Python and Streamlit that supports Text-to-Text, Speech-to-Text, and Speech-to-Speech translation across 100+ languages.

This project combines language translation, speech recognition, and text-to-speech technologies into a single user-friendly platform to help break communication barriers globally.

🚀 Features
🌐 Translate text into 100+ languages
🎤 Speech-to-Text conversion
🔊 Speech-to-Speech translation
🧠 AI-powered language processing
🎧 Audio playback for translated speech
💻 Modern and interactive Streamlit UI
⚡ Fast and lightweight system
🛠️ Technologies Used
Python
Streamlit
deep-translator
SpeechRecognition
gTTS
📌 Modules
app.py

Main Streamlit user interface

translator.py

Handles multilingual text translation

speech.py

Handles speech recognition and audio generation

languages.py

Contains language mappings for 100+ languages

⚙️ Installation

Clone the repository:

git clone YOUR_GITHUB_REPO_LINK
cd AI-Multi-Language-Translator

Create virtual environment:

python -m venv venv

Activate virtual environment:

.\venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Run the project:

streamlit run app.py
